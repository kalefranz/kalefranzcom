ErwinJr is a software suite for designing quantum cascade lasers and other intersubband heterostructure devices. It is fast; highly optimized numerical solutions are implemented in C++, while the flexibility of the Matlab scripting language is used for the graphical user interface and array handling. Current supported materials systems include InGaAs/AlInAs (with multiple material compositions in a single structure), GaAs/AlGaAs, and ZnCdSe/ZnCdMgSe. The software is highly and easily customizable, so new features can be quickly added.

This software is free for most academic use. For-profit use requires licensing through the Princeton Univeristy Office of Technology Licensing and the Princeton University Mid-Infrared Photonics Group. Contact me or Prof. Claire Gmachl for more information.
